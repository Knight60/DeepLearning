ข้อสังเกต

  • CNN-2D แม่นสุดในการวัดแบบจุด (76.9% OA) เพราะ patch 9×9 ให้บริบทรอบจุดพอดี โดยไม่เจือจาง label เหมือน U-Net
  • U-Net ฝึกแบบ point-supervised — มี label แค่จุดเดียวต่อ patch 32×32 (สลับ weight=1 เฉพาะพิกเซลกลาง, 0 ที่เหลือ) เพราะข้อมูลจาก BigQuery เป็น point label ไม่ใช่ dense mask ผลคือ accuracy ใกล้เคียง CNN-2D (76.1%) แต่ใช้ข้อมูลน้อยกว่ามากต่อการอัพเดตหนึ่งครั้ง
  • แผนที่ทั้งพื้นที่ของ U-Net ต่างจากอีกสองโมเดลชัดเจน โดยเฉพาะฝั่งตะวันออกที่ทำนาย "น้ำ" มากกว่าที่ CNN-1D/2D เห็น — เป็นสัญญาณว่าโมเดล dense-prediction ที่ฝึกจากจุดตัวอย่างน้อย (800 จุด) generalize ไปยังพื้นที่ที่ไม่มีจุดฝึกใกล้เคียงได้ไม่เท่ากับโมเดลที่ทำนายทีละจุด ควรเพิ่มจุดตัวอย่างให้กระจายทั่วพื้นที่ก่อนใช้แผนที่ U-Net นี้จริงจัง
  • คลาส 3 (เกษตรกรรม) เป็นจุดอ่อนร่วม ของทั้งสามโมเดล (producer's accuracy 59.6–66.7%) น่าจะมาจากความหลากหลายของพืชในพื้นที่ที่ label เดียวไม่ครอบคลุมสเปกตรัมที่ต่างกัน
